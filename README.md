# For local developement
1. install xampp
2. Go to .env file and change the DB connection
3. run <b>usmansaleem234_lyskills_new.sql </b> and <b>user_ann_models.sql</b> file locally
4. Add primary key manually in every table or create a alter query and try changing table name and primary key column
5. composer install
6. npm install
7. RUN php artisan serve --port=8081
